Parenting: It's a Life

Web development case study of sewagefreeseas.org for educational purposes. My goal was to recreate the desktop layout of the site from scratch using HTML and CSS, without viewing or using any of the original source code. The only resources used from the original site are the image sprite, the hero image and the background texture image.

I recreated the sliding menu effect with CSS animations and a simple click handler. I also learned how to use CSS with image sprites to place the logos and social buttons. Typefaces are approximated using free options from Google Fonts.
